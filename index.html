Introdução

Bem-vindo ao Guia Completo de Ambiente de Desenvolvimento Linux

Este tutorial foi criado para ajudar você a montar um ambiente de desenvolvimento completo em Linux, utilizando o Oracle VirtualBox e Ubuntu 22.04 LTS. Seja você um iniciante sem experiência prévia ou um desenvolvedor experiente, este guia fornecerá instruções detalhadas para configurar um ambiente versátil e poderoso.

O que você vai aprender:

  • Instalação e configuração do Oracle VirtualBox
  • Criação de uma máquina virtual com Ubuntu 22.04 LTS
  • Configuração pós-instalação do Ubuntu
  • Instalação e configuração de Python, Node.js, Docker e PostgreSQL
  • Configuração do VS Code e Cursor AI
  • Integração com ferramentas modernas de IA (Claude API, DeepSeek SDK, OpenAI)
  • Organização de projetos e boas práticas de desenvolvimento

Requisitos

Para seguir este tutorial, você precisará de:

  • Um computador com pelo menos 8GB de RAM (16GB recomendado)
  • Pelo menos 50GB de espaço livre em disco
  • Processador com suporte à virtualização (Intel VT-x ou AMD-V)
  • Conexão com a internet

Como usar este tutorial

Este tutorial está organizado em módulos sequenciais. Recomendamos seguir a ordem apresentada, pois cada módulo se baseia no anterior. Use o menu de navegação para acessar cada seção.
